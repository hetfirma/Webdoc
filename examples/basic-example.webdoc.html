Basis Webdoc Voorbeeld

Door Jan Jansen

Gepubliceerd op 16 oktober 2025

Introductie

Dit is een eenvoudig voorbeeld van een Webdoc document. Webdoc is een open, uitbreidbaar documentformaat dat gebaseerd is op webstandaarden zoals HTML5, JSON-LD en RDFa.

Dit document demonstreert de basis features:

  • HTML5 voor structuur en inhoud
  • JSON-LD voor metadata
  • RDFa voor semantische annotaties
  • CSS3 voor styling

Metadata

Het document bevat gestructureerde metadata in JSON-LD formaat in de <head> sectie. Deze metadata bevat informatie over de auteur, publicatiedatum, versie en trefwoorden.

Semantische Annotaties

De inhoud is geannoteerd met RDFa attributes zoals typeof en property. Dit maakt de inhoud machine-leesbaar en geschikt voor geautomatiseerde verwerking.

Styling

Het document bevat embedded CSS styling die gebruikmaakt van moderne CSS features zoals custom properties (CSS variabelen) en is responsive voor verschillende schermformaten.

Self-Contained

Dit document is volledig self-contained. Het heeft geen externe afhankelijkheden en kan direct geopend worden in elke moderne webbrowser.
